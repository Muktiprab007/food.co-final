# Food.co

Food.co is a robust and scalable food catalog/ordering web application. This project focuses on high-quality code standards, modularity, and comprehensive testing to ensure a seamless user experience.

## 🚀 Key Features
* **Modular Architecture**: Built using ES6 Modules and bundled with Webpack.
* **Production Optimized**: Includes separate configurations for development and production environments.
* **Quality Assurance**: Integrated with ESLint for code linting and Babel for cross-browser compatibility.
* **Automated Testing**: 
    * **Unit Testing**: Powered by Jest.
    * **End-to-End (E2E) Testing**: Powered by CodeceptJS.

## 🛠️ Tech Stack
* **Language**: JavaScript (ES6+)
* **Bundler**: Webpack
* **Transpiler**: Babel
* **Testing Frameworks**: Jest & CodeceptJS
* **Linting**: ESLint
